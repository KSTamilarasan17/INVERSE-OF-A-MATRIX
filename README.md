To write a python program to find the inverse of a matrix

### Step 4: 
End the program

## Program:
```
#Program to find the inverse of a matrix.
#Developed by:TAMILARASAN K S.

import numpy as np
A=[[1,0,3],[-1,2,-2],[2,3,-1]]
inv= np.linalg.inv(A)
print(inv)
```
